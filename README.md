# martjn.net

Interactive SQL portfolio website — query my profile, dive logbook, and blockchain data via a terminal-style SQL interface.

**Live:** [https://martjn.net](https://martjn.net)

## Features

- Custom SQL engine in vanilla JS (SELECT, WHERE, ORDER BY, LIMIT, GROUP BY, aggregates)
- No external dependencies — single HTML file
- Dynamic blockchain data (block heights, timestamps, mempool)
- Scuba & freediving logbook
- Keyboard navigation (ESC for menu mode, arrow keys, Tab autocomplete)
- Mobile-friendly

## Usage

Type `HELP` for available commands or click the suggested queries below the terminal.
